How to use

  1. Click each Download button above to save the PNG files.
  2. Move icon-192.png and icon-512.png into public/icons/.
  3. Reference them in your manifest.json: 
    {
  "icons": [
    {
      "src": "/icons/icon-192.png",
      "sizes": "192x192",
      "type": "image/png",
      "purpose": "any maskable"
    },
    {
      "src": "/icons/icon-512.png",
      "sizes": "512x512",
      "type": "image/png",
      "purpose": "any maskable"
    }
  ]
}
